Various structs for Respawn's models including ones unchanged from normal Source Engine (V48 & V49)

Valve:

  -V47 (with og xbox version lul)  
  -V48  
  -V49  

Respawn:

  -V52  
  -V53  
  -V54 subversions: 8/9/10/11/12/12.1/12.2/13/14(.1?)
  
As well as:
  
  -VPHY   
  -RPHY (incomplete)  
  -VTX subversions: 7
  -VVD  
  -VVC (partial)  
  -VG (soon)
